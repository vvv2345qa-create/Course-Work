Проект предназначен для анализа Apache `access.log` и автоматического выявления подозрительных IP-адресов.  
В качестве основного алгоритма используется `Isolation Forest`, который определяет аномальное поведение пользователей на основе признаков, извлеченных из логов и дополнительно соотносит с двумя классами (сканерами и ботнетами).

В проекте выполняется:

- парсинг Apache access.log;
- группировка событий по IP-адресам;
- нормализация URL-путей;
- разбиение URL на n-граммы;
- формирование матрицы признаков;
- обучение модели Isolation Forest;
- сохранение и загрузка обученной модели;
- проверка тестовой выборки;
- идентификация двух классов (сканеров и ботов)
- отображение результатов в графической оболочке.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск анализа через консоль

```bash
python access_log_features.py
```

## Запуск графической оболочки

```bash
python access_log_gui.py
```

## Основные файлы проекта

- `access_log_features.py` - основная логика обработки логов, обучения и проверки модели;
- `access_log_gui.py` - графическая оболочка приложения;
- `model_metrics.py` - расчет метрик качества и построение матрицы ошибок;
- `requirements.txt` - список зависимостей проекта.
